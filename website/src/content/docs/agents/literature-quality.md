The literature quality agent decides which papers should be included in the final review.

## What it does

It scores papers using relevance to the review question, recency, and influence signals, then produces kept/rejected sets with explicit reasons.

## Output

A scored inclusion table and filtering rationale used by the literature synthesizer.
